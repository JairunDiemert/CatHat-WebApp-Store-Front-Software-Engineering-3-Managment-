# Mad Hatters

To run our project:

<b>Turn on Service</b>
  1. cd into our server folder
  2. run npm install
  3. node index.js
  
<b>Turn on Gateway</b>
  1. cd into our api-gateway folder
  2. run npm install
  3. node api-index.js

<b>Then Turn on Client</b>
  1. cd into our cat-hats folder
  2. run npm install
  3. ng serve or npm start
  4. Go to localhost:4200
