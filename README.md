# Mad Hatters

To run our project:

1. Turn on Service
  1. cd into our server folder
  2. run npm install
  3. node index.js
  
2. Turn on Gateway
  1. cd into our api-gateway folder
  2. run npm install
  3. node api-index.js

3. Turn on Client
  1. cd into our cat-hats folder
  2. run npm install
  3. ng serve or npm start
  4. Go to localhost:4200
